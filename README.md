# Android Project

### 用 java 撰寫各種內嵌式系統設計專案
* **Final Project - APP 源遊**  
	 * 新增開起app的動畫，增加app美感
   * 以「綠建築」為主題做出app，提供使用者更快查詢綠能相關建築
   * 可多頁面切換，以不同主題為切分，也可選擇建築區域，讓使用者可以依照感興趣的部分深入了解
   * 首頁提供動態切換的綠能介紹圖片，增加使用者的綠能資訊觸及率
   * 點擊首頁切換的圖片有對話框顯示更多綠能的資訊，並有網站連結可讓感興趣的使用者深入了解相關資訊，以推廣綠能知識
